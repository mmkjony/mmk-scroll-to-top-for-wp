A super simple, highly customizable and light-weight plugin that adds a flexible scroll back to top button to your wordpress site

== Description ==
The easiest way to have the ability to add a flexible \"Scroll to Top\" button to your website. This button automatically appears at the bottom-right corner of the website while scrolling down a page on your website. And, when someone click on this button, it will let the web page to smoothly scroll back to the top in an elegant way. You can use this without any modification of your template.

**Features:**
* Smooth scrolling animation, powered by jQuery
* Compatible with all WordPress themes and plugins
* Highly customizable options via the admin panel
* Cross Browser Compatible (works on almost all browsers)
* No need to add any markup to your theme

== Installation ==
You can install this plugin easily through the following steps:

**Automatically via Wordpress:**
* Navigate to the \"Plugins\" page in your WordPress admin panel.
* Click on \"Add New\" and search for \"MMK Scroll Back To Top\".
* Download and activate.

**Install Manually:**
* Download the plugin files.
* Unzip and copy/upload the plugin folder to your WordPress Content/plugins directory.
* Navigate to the \"Plugins\" page in your WordPress admin panel. The plugin will be displayed in your plugins list.
* Click activate.

You will now have a sub-menu inside the settings option in Wordpress admin panel to customize the plugin.

== Frequently Asked Questions ==
= Will this Plugin work on my Wordpress.com website? =
Sorry, this plugin is available for use only on self-hosted (WordPress.org) websites.

= Does this plugin require modification to the theme? =
Absolutely not. This plugin can be customized entirely from the Wordpress admin panel.

= How to customize this plugin? =
After activating this plugin, you will find a sub-menu named \"MMK Scroll To Top\" inside \"Settings\" menu in your Wordpress admin panel.
